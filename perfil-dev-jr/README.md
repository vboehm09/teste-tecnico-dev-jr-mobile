# Perfil Dev Jr

## Descrição
Aplicativo criado em React Native com Expo para apresentar um perfil profissional de candidato a Desenvolvedor Mobile Júnior. O foco do projeto é transformar ideias em telas simples, funcionais e fáceis de navegar, demonstrando domínio prático na construção de interfaces.

## Tecnologias utilizadas
- React Native
- Expo
- JavaScript
- React Navigation
- Git
- GitHub

## Funcionalidades
- Exibe dados do candidato
- Apresenta uma tela de perfil
- Lista habilidades
- Mostra projetos
- Exibe informações de contato
- Possui navegação por abas inferiores

## Telas
- **Perfil:** Apresenta uma introdução sobre mim, destacando meus objetivos como desenvolvedor em início de carreira, meu interesse por acessibilidade e minha motivação em criar aplicativos claros e organizados.
- **Habilidades:** Lista as principais tecnologias e conceitos que domino, focando nos fundamentos do desenvolvimento mobile.
- **Projetos/Contato:** Exibe os trabalhos desenvolvidos e disponibiliza informações diretas para contato profissional (e-mail, LinkedIn, GitHub).

## Organização do código
- **Separação de responsabilidades:** A aplicação foi organizada de forma modular, com as telas separadas em arquivos distintos para facilitar a manutenção e escalabilidade do projeto.
- **Uso de componentes nativos:** O código foi construído utilizando os componentes básicos (core) do React Native, garantindo uma estrutura limpa e direta.

## Diferenciais
- **Navegação otimizada e customizada:** A principal decisão técnica do projeto foi a criação da navegação por abas inferiores feita de forma leve e totalmente customizada, sem depender de bibliotecas visuais prontas de UI. Isso demonstra domínio sobre os fundamentos do React Native e preocupação real com a performance do aplicativo.
- **Foco em UX (Experiência do Usuário):** Priorizei uma interface sem poluição visual, onde a usabilidade é o centro da experiência, provando que é possível entregar um ótimo design apenas dominando a base da tecnologia.

## Como executar o projeto
1. Clone este repositório em sua máquina:
   `git clone https://github.com/vboehm09/teste-tecnico-dev-jr-mobile.git`
2. Acesse a pasta do projeto no seu terminal:
   `cd teste-tecnico-dev-jr-mobile`
3. Instale as dependências do projeto:
   `npm install`
4. Inicie o servidor do Expo:
   `npx expo start`
5. Abra o aplicativo no seu dispositivo físico lendo o QR Code com o app **Expo Go**, ou pressione `a` no terminal para rodar em um emulador Android / `i` para simulador iOS.
